# ייצוא דף המכירה ל-WordPress

דף המכירה מיוצא כ־HTML סטטי בשלושה חלקים, כדי לאפשר החדרת Widget טופס הצ'קאאוט ביניהם.  
**דף האפסאל (upsell) הוא עמוד נפרד** – יש לו קבצי HTML משלו (למטה) ואין לכלול אותו בתוך דף המכירה.

## סדר ההרכבה בעמוד (דף המכירה בלבד)

1. **sales-page-global.html** – להכניס **פעם אחת** (למשל ב־Header או ב־HTML הראשון בעמוד):
   - פונט Rubik, משתני CSS (`:root`), מחלקת `.marker` להדגשת מילים, Tailwind CDN + הגדרות theme.
   - **Overlay מחסר**: ב־95% גלילה או אחרי 7 דקות בדף מופיע מסך — קודם "עצרי!" 2 שניות, אחר כך תוכן + קוד קופון **CARE10** (10% הנחה) וכפתור "ברגע שתלחצי כאן / יש לך 5 דקות בדיוק לממש את ההטבה". אין שמירה בין כניסות (כל טעינה = התנהגות כמו בפעם הראשונה).
   - **טיימר + ויגנט**: אחרי לחיצה על הכפתור מופיעים ויגנט אדום מהבהב וטיימר "הקופון: CARE10" / "בתוקף עד" **5:00** (ספירה לאחור של 5 דקות).
   - **סקריפט**: מטפל גם ב־CTA הדביק — מופיע אחרי גלילה של 30% ונשאר גלוי (לא מוסתר בחזרה).

2. **sales-page-before-checkout.html** – התוכן עד טופס ההרשמה:
   - Hero (תמונת גיבור + 2 ביקורות) → בעיה → פתרון (2 ביקורות) → כאב → אויב → מדע → ההצעה (2 ביקורות) → סגירה 21 יום → תמחור (297 ₪) + תמונות ביקורת מפוזרות.
   - אין תיבות אפורות (placeholder) – תמונות הביקורת מקושרות לכתובות מ־stayyoung.live.
   - תמונת הגיבור ב־Hero: מפנה ל־`https://www.stayyoung.live/wp-content/uploads/2026/02/2size-jeans-image.png` (אין צורך בהחלפה).
   - מסתיים בהערה: `<!-- כאן ייכנס Widget טופס הצ'קאאוט -->`

3. **Widget טופס הצ'קאאוט** – הטופס של וורדפרס / מערכת התשלומים.

4. **sales-page-after-checkout.html** – התוכן אחרי הטופס:
   - איך זה עובד (4 שלבים) + בלוק 6 תמונות ביקורות
   - אם יש משהו שמונע ממך להצטרף + תמונת ביקורת
   - אודות אביב: תמונת אביב (aviv-bellyfat.png) מתחת לכותרת "אביב, נעים להכיר:" + תמונת ביקורת בסוף
   - CTA דביק: רק כפתור (ללא טקסט משמאל), רוחב מלא (100%), טקסט "יאללה אביב - תוריד אותי ב2 מידות בג׳ינס!" (קישור ל־`#pricing`)

## דף האפסאל (עמוד נפרד)

דף האפסאל **לא** כלול בקבצי דף המכירה. יש להשתמש בקבצים הנפרדים הבאים **בעמוד וורדפרס אחר** (למשל אחרי תשלום / thank-you):

1. **upsell-page-global.html** – להכניס **פעם אחת** בראש העמוד:
   - פונט Rubik (כולל כפייה על `main.upsell-page-rubik` ו־`div.upsell-page-rubik`), משתני CSS, `.marker`, Tailwind, Bold→600. אנימציית הקלדה עם צליל (90 שניות ברירת מחדל). ללא overlay מחסר וללא טיימר.

2. **upsell-page-content.html** – תוכן העמוד:
   - תודה שהצטרפת → הצטרפת לקורס בהצלחה → אימייל עם כל הפרטים.
   - מסר מאביב...: אנימציית הקלדה (חכי…, יש לך מסר אישי מאביב, הדרכה חד פעמית, אפקט הוואו, 2 כלים ספציפיים, 87 ש״ח, תוך 3 דקות, מרגע הלחיצה על הכפתור...) — תואם ל־UpsellTypingSection ב־React, כולל הדגשות (.marker) ו־spacers.

## פונט Rubik (כנגד תבנית האתר)

- דף המכירה מגדיר פונט **Rubik** (גם ב־CSS וגם בסקריפט שמריץ כפייה אחרי טעינה).
- אם תבנית וורדפרס עדיין מציגה את פונט האתר: ב־**עיצוב → התאמה אישית → CSS נוסף** הוסף:
  ```css
  main.sales-page-rubik, main.sales-page-rubik *,
  div.sales-page-rubik, div.sales-page-rubik *,
  #scarcity-overlay, #scarcity-overlay *, #sticky-cta-bar, #sticky-cta-bar * {
    font-family: 'Rubik', Arial, sans-serif !important;
  }
  ```
  כך הכללים נטענים אחרון ו-Rubik יישאר על דף המכירה.
- לדף האפסאל: ב־CSS נוסף אפשר להוסיף `main.upsell-page-rubik, main.upsell-page-rubik * { font-family: 'Rubik', Arial, sans-serif !important; }` אם התבנית משתלטת.

## הערות

- כל החלקים עם `dir="rtl"` ו־`lang="he"`.
- ה־CTA הדביק מוסתר בהתחלה ומופיע אחרי גלילה של 30% ונשאר גלוי (הסקריפט ב־global מטפל בזה). רוחב הכפתור: 100%.
- Overlay וטיימר פועלים רק אם ה־global נטען פעם אחת באותו עמוד.
- תמונות הביקורת (testimonials) ותמונת הגיבור ב־Hero מפנות ל־stayyoung.live (גיבור: 2size-jeans-image.png).
